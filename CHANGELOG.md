## [1.0.0] - 2026-07-20

### Added
- **Authentication System**
  - User registration and login
  - Password reset functionality
  - Role-based access control

- **Field Report Form**
  - Multi-step form with 8 sections
  - Focal person details
  - Training centre information
  - Attendance tracking
  - Employment outcomes
  - Training quality assessment
  - Partner engagement tracking
  - Safeguarding checklist
  - Document and media uploads

- **Report Management**
  - Save drafts locally
  - Submit reports to Firebase
  - View report history
  - Edit and delete reports
  - Search and filter reports

- **Email Integration**
  - Automatic email to CEO
  - Email to registration officer
  - Confirmation email to sender
  - PDF report attachments

- **Google Sheets Integration**
  - Automatic data sync to Google Sheets
  - Central database for all reports
  - Easy data analysis and reporting

- **Offline Support**
  - Save reports locally when offline
  - Auto-sync when connection restored
  - Draft management

- **Media Capture**
  - Camera integration for photos
  - Video recording capability
  - Document scanning
  - Image compression

- **User Interface**
  - Material Design 3
  - Dark mode support
  - Responsive layout
  - Form validation
  - Progress indicators

- **Security**
  - Firebase Authentication
  - Data encryption
  - Secure API key storage
  - Role-based permissions

- **Reporting**
  - PDF report generation
  - Data export capabilities
  - Statistics dashboard
  - Quick stats overview

### Technical
- Flutter 3.x
- Dart 3.x
- Firebase integration
- Provider state management
- Hive for offline storage
- PDF generation
- Camera and media handling
